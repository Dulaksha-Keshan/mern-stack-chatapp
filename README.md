## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Dulaksha-Keshan/mern-stack-chatapp.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd mern-stack-chatapp
   ```

3. **Install dependencies and build the app**
   
   ```bash
   npm run build
   
4. **Start the app**

    ```bash
     npm start
